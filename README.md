# 📊 Excel Sales Dashboard

This project is an interactive Excel dashboard built to analyze sales data using Pivot Tables, KPIs, and visual charts.

## 🔹 Project Overview
The dashboard provides insights into:
- Total Sales and Profit
- Sales by Category
- Sales by City
- Monthly Sales Trend
- Top Salesperson Performance

## 🔹 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization
- Slicers & Filters

## 🔹 Key Features
- Dynamic filtering using slicers (City & Category)
- KPI summary (Total Sales, Profit, Average, Max, Min)
- Interactive charts
- Clean and structured data model

## 🔹 Insights
- Best Category: Electronics
- Top City: Amman
- Top Salesperson: Omar
- Lowest Category: Stationery

## 📁 File Included
- Excel_Project_Model_Answer.xlsx

## 🚀 How to Use
1. Download the Excel file
2. Open in Microsoft Excel
3. Use slicers to interact with the dashboard
